# Team Management System

A modern web-based team management system built with HTML, CSS, and JavaScript, featuring 3D elements and interactive components.

## Features

- Interactive 3D background using Three.js
- Team member cards with 3D flip effect
- Task management system with local storage
- Progress tracking
- Responsive design
- Modern UI with glassmorphism effects

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid, Flexbox, and CSS Variables)
- JavaScript (ES6+)
- Three.js for 3D effects
- Local Storage for data persistence

## Getting Started

1. Clone the repository
2. Open `index.html` in a modern web browser
3. No build process or dependencies required

## Features in Detail

### 3D Background
- Interactive particle system
- Responsive to window resizing
- Smooth animations

### Team Member Cards
- 3D flip effect on hover
- Displays member information
- Responsive grid layout

### Task Management
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Persistent storage using localStorage

### Progress Tracking
- Visual progress bars
- Real-time updates

## Browser Support

The application works best in modern browsers that support:
- CSS Grid
- CSS Flexbox
- CSS Variables
- ES6+ JavaScript features
- WebGL (for Three.js)
